# Aerodrome Finance
Aerodrome Finance is a highly advanced decentralized exchange (DEX) and automated market maker (AMM) operating as the foundational liquidity infrastructure for the Coinbase-incubated Base Layer-2 blockchain. Founded in August 2023, Aerodrome functions as a comprehensive "MetaDEX," inheriting the battle-tested smart contract architecture of Velodrome V2 while pioneering advanced features such as the Slipstream concentrated liquidity module.

## Start building with Aerodrome Finance

Aerodrome Finance provides developers with a highly efficient, permissionless, and immutable liquidity engine optimized specifically for the high-throughput Base network. By leveraging the protocol's deeply liquid architecture, engineers can rapidly construct specialized decentralized finance applications, automated yield optimization strategies, DAO governance tooling, and advanced algorithmic trading dashboards. ### Libraries
* **Aerodrome Smart Contracts**: The core Solidity implementations containing the immutable AMM mechanics, the voting escrow mathematical logic, and the factory deployments. [(Explore Contracts)](https://github.com/aerodrome-finance/contracts)  
* **Slipstream Concentrated Liquidity**: The highly specialized smart contract logic for providing customized, concentrated liquidity ranges and dynamic fee algorithms up to 3%. [(Explore Slipstream)](https://github.com/aerodrome-finance/slipstream)  

* **Sugar SDK**: An essential on-chain data aggregation layer and associated SDK that empowers developers to retrieve comprehensive, paginated liquidity pool states in a structured format, vastly simplifying dApp state management. [(Explore Sugar)](https://github.com/velodrome-finance/sugar) 
* **Relay Automation Protocol**: A robust suite of smart contracts engineered to fully automate the compounding, management, and interaction logic of $veAERO NFTs. [(Explore Relay)](https://github.com/aerodrome-finance/relay)
