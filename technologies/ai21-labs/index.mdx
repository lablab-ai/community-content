# AI21 Labs

AI21 Labs is a platform for building AI applications. It provides a suite of tools and guides
for building AI applications and training models. AI21 Labs helps developers and businesses use
cutting-edge language models, with no NLP expertise required. Access powerful language models using
an API that makes you smile

### AI21 Studio

AI21 Studio is a platform for building AI applications that comprehend and generate natural language, powered by AI21 Labs' cutting-edge language models

- [AI21 Studio Home](https://studio.ai21.com/overview) Hit the ground running with pre-made examples or build your own
- [AI21 Studio Playground](https://studio.ai21.com/playground/) Playground is a place to experiment with AI21 Studio's language models
- [AI21 Studio Documentation](https://docs.ai21.com/) Documentation for AI21 Studio
- [AI21 Studio API Reference Sandbox](https://docs.ai21.com/reference/completion) API Reference Sandbox for AI21 Studio
- [AI21 Studio Prompt Engineering](https://docs.ai21.com/docs/prompt-engineering) Learn prompt engineering for LLMs
- [AI21 Studio Custom Models](https://docs.ai21.com/docs/custom-models) Custom models are fine-tuned for optimal performance on a training set of examples representing a specific task.
- [AI21 Studio Recipes](https://docs.ai21.com/recipes) Interactive guides to help you get started with different concepts and tasks
