## **Guide to Access Llama 270-B with Langchain**

[Access Llama 2(70B) model from Clarifai with Langchain - Video](https://youtu.be/N0jVQlzWMrs)

1. **Installation**
    
    Before anything else, you need to have both [Clarifai](https://www.clarifai.com/products/platform?utm_campaign=hack&utm_medium=REF&utm_source=lablab) and [Langchain](https://lablab.ai/tech/langchain) installed on your device.
    
2. **Log in & Obtain Personal Access Token**
    - Log in to your Clarifai account.
    - Navigate to 'Settings' followed by 'Security.'
    - Here, you can obtain your personal access token. If you don’t have one, simply click on ‘Create Personal Access Token.’
    - Provide a brief description and select the scope of your intended use.
    - Copy your personal access token once it’s generated. This token is your key to accessing resources within the Clarifai platform.
3. **Integrate with Langchain**
    - Import Clarifai into Langchain from the Llm's module.
    - Initialize it using the personal access token and the specific details of the model you aim to use.
    - For our current endeavor, we'll be using the LAMA 270-B model. To do so, specify the user ID, app ID, and model ID. You can easily find these details under the 'Model' section of Clarifai.
    - Once you’ve filled in the details, run the code.
4. **Utilizing the Model**
    - Use the initialized model to process text or data.
    - Pass the desired text through the model and await the response.
5. **Expand Your Horizons**
    
    While we’ve solely focused on the LAMA 270-B model in this tutorial, remember that the Clarifai platform is brimming with numerous models. You can access and use them similarly, opening doors to a plethora of AI-powered applications.
